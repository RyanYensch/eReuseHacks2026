# AI Detector Extension

This UNIHACK 2026 project is a Chrome extension that allows users to detect if different content on the web page they are on is likely to be AI.
This includes:

- Text
- Images
- Videos

## How To Run

### Extension

```BASH
cd browser-ai-detector
npm i
npm run dev
```

### Backend

```BASH
cd backend
npm i
npm run dev
```
